# Introduction
This folder contains the following database schemas as csv files. The details have been listed in the description below.

### Simplified Linear Model Dataset Schema
This schema contains the structural format for the dataset generated for the simplified linear model in a csv file. Simply upload this file when executing the simplified linear model dataset generator to get a dataset in csv file.

### Unicycle Model Dataset Schema
This schema contains the structural format for the dataset generated for the unicycle model in a csv file. Simply upload this file when executing the unicycle model dataset generator to get a dataset in csv file.
